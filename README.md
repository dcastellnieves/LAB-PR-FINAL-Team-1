An application with an interactive GUI that lets you design your own mazes. You can choose from the available algorithms to search and find the shortest path from your desired start-to-end point.

![Interactive Maze](https://media.giphy.com/media/mrxLQ1QGOIKJGgUyQG/giphy.gif)

---

##  Features

* **Included Algorithms:**
    * Breadth-First Search (BFS)
    * Depth-First Search (DFS)
    * A\* Search
* **Save & Load:** Options to save or load mazes. You can also find sample mazes to load in the project directory.
